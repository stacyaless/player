# Music Player - 现代化音乐播放器

一个基于Python和customtkinter构建的现代化音乐播放器应用，具有优雅的UI界面和丰富的功能。

##  主要特性

-  **现代化UI设计** - 使用customtkinter实现Material Design风格界面
-  **拖拽添加** - 支持拖拽音频文件（MP3、WAV、FLAC、M4A）到窗口
-  **实时歌词** - Apple Music风格的滚动歌词显示
-  **背景动画** - 播放时背景图片动态偏移效果
-  **播放列表管理** - 下拉菜单形式，支持选择和删除
-  **在线资源获取** - 实验性功能，通过音乐API获取歌词和封面
-  **本地缓存** - 自动缓存网络获取的资源
-  **进度控制** - 支持进度条拖动和点击跳转
-  **播放控制** - 播放/暂停、上一首/下一首功能

## 项目结构

```
player-main/
├── main.py              # 主入口文件，包含完整的音乐播放器GUI
├── config.py            # 配置文件，存储应用程序参数
├── utils.py             # 工具模块，提供通用函数
├── assets.py            # 资源模块，动态生成UI图标
├── metadata.py          # 元数据处理模块，读取音频文件元数据
├── online_fetcher.py    # 在线获取模块，通过网络获取歌词和封面
├── splash.py            # 开屏页面
└── app_icon.ico         # 应用程序图标
```

## 安装和运行

### 前提条件

- Python 3.7+
- pip 包管理器

### 安装依赖

```bash
pip install customtkinter pygame pillow mutagen tinytag tkinterdnd2 requests urllib3
```

### 运行应用

```bash
python main.py
```

##  模块说明

### main.py - 主入口文件
- **核心类**: `MusicPlayer`（继承自ctk.CTk和TkinterDnD.DnDWrapper）
- **主要功能**:
  - 拖拽添加音乐文件
  - 音乐播放控制
  - 进度条操作
  - 实时歌词显示
  - 播放列表管理
  - 背景动画效果

### config.py - 配置文件
- **功能**: 存储应用程序的配置参数
- **配置项**:
  - 窗口初始尺寸（400x800）
  - 布局比例
  - 歌词显示设置
  - 颜色配置

### metadata.py - 元数据处理模块
- **功能**: 读取音频文件的元数据
- **支持的格式**: MP3、FLAC、M4A等
- **主要功能**:
  - 获取歌曲信息
  - 读取内嵌封面图片
  - 解析LRC格式歌词

### online_fetcher.py - 在线获取模块
- **功能**: 实验性功能，通过网络获取歌词和封面
- **数据源**: 音乐API
- **特性**:
  - 本地缓存机制
  - 错误处理和超时机制
  - 需要本地部署音乐API

## 使用说明

1. **添加音乐**:
   - 直接拖拽音频文件到播放器窗口
   - 点击"导入"按钮选择文件

2. **播放控制**:
   - 使用播放/暂停按钮控制播放状态
   - 使用上一首/下一首按钮切换歌曲
   - 拖动进度条跳转到指定位置

3. **歌词显示**:
   - 自动加载本地LRC歌词文件
   - 实时滚动显示当前播放行
   - 支持在线获取歌词（需要API）

4. **播放列表**:
   - 使用下拉菜单查看所有歌曲
   - 点击选择要播放的歌曲
   - 支持删除不需要的歌曲

## 🔧 配置选项

在`online_fetcher.py`中可以自定义api在线调取封面、歌词：

```python
self.netease_api = "http://yourip:3000"  

```
## 注意事项

1. **在线功能**: 在线获取歌词和封面需要本地部署音乐API
2. **性能**: 背景动画可能会对性能有一定影响，可在配置中调整
